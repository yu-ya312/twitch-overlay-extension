# Twitch Chat Overlay Extension


## 概要

Twitchで配信をフルスクリーン視聴する際にチャットが見えなくなる問題を解決するChrome拡張です。

チャットを画面上にオーバーレイ表示できます。



## 作成背景

Twitchで配信をフルスクリーンで視聴するとチャット欄が表示されなくなることに不便を感じたため、

視聴しながらチャットも確認できるようにする目的で作成しました。



## 主な機能

・チャットのオーバーレイ表示

・表示のON / OFF切り替え

・ドラッグによる位置変更

・サイズ変更機能


## 使い方
1. Chromeで拡張機能を開く
2. 「デベロッパーモード」ON
3. フォルダを読み込む


## インストール方法

1. このリポジトリをダウンロード
2. Chromeで「chrome://extensions/」を開く
3. デベロッパーモードON
4. 「パッケージ化されていない拡張機能を読み込む」
5. フォルダを選択
