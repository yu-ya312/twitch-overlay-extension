# Twitch Chat Overlay Extension

Twitchをフルスクリーンでもチャット表示できるChrome拡張

## 機能
- 半透明チャット表示
- 位置調整
- ON/OFF切替


## 使い方
1. Chromeで拡張機能を開く
2. 「デベロッパーモード」ON
3. フォルダを読み込む


## インストール方法

1. このリポジトリをダウンロード
2. Chromeで「chrome://extensions/」を開く
3. デベロッパーモードON
4. 「パッケージ化されていない拡張機能を読み込む」
5. フォルダを選択
